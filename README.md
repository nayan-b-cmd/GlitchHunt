# GlitchHunt 

Live pricing glitch and deal tracker for India and US retail stores.

## What it does

- Browse curated pricing glitches and great deals across Zara, H&M, Myntra, Marshalls, TJ Maxx, and more

- Search any item — AI hunts for live glitches and deals using real-time web search

- Paste any deal into the Verify tab to check if it's a real glitch, a great deal, or suspicious

- Save deals to a personal watchlist

## How it works

Single HTML file. Uses the Anthropic Claude API with web search to find and classify deals in real time.

## Stack

Vanilla HTML · CSS · JavaScript · Anthropic Claude API · Tabler Icons
